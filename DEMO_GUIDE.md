## Instant Access - No OTP Required!

The application is now in demo mode for easy design showcase and testing. You can instantly access both Admin and Dealer portals with a single click.

## Quick Start

When you open the application, you'll see a beautiful landing page with two options:

### 1. Admin Panel
- **Click**: "Login as Admin" button
- **Access**: Full administrative control
- **Features**:
  - Approve/reject dealer registrations
  - Manage all orders (approve, reject, complete)
  - Configure live MCX pricing
  - Create and manage discount coupons
  - Send notifications to dealers

### 2. Dealer Portal
- **Click**: "Login as Dealer" button
- **Access**: Complete dealer experience
- **Demo Account**: Silver Star Jewellers
- **Available Credit**: ₹85,000
- **Features**:
  - Browse 6 silver products
  - Add items to cart with size/quantity
  - View real-time pricing calculations
  - Apply discount coupons (WELCOME10, FESTIVE500)
  - Place orders with online/credit payment
  - Track order history (3 sample orders included)
  - View profile and credit information
